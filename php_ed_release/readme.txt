Shortcuts:
-------------------------
Ctrl+O    Open
Ctrl+S    Save
Ctrl+W    Close File
Ctrl+G    Go to line
Ctrl+F    Find
F3        Find Next 
Ctrl+K    Color selector
Ctrl+D    Go to definition
Ctrl+I    Open file at cursor
Ctrl+E    Insert "echo " or "<?php echo  ?>"
Alt+Up    Previous file
Alt+Down  Next file
Esc       Close auto-complete window
Alt+F4    Close program


Auto-complete:
-------------------------
objects,  "->"
classes,  "::"
arrays,   "['"
variables <by name>
functions <by name>
constants <by name>
keywords  <by name>


Features:
-------------------------
Function argument, javadoc preview
Tab in auto complete window inserts first common substring
