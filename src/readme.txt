Shortcuts:
-------------------------
Ctrl+O    Open
Ctrl+S    Save
Ctrl+W    Close File
Ctrl+G    Go to line
Ctrl+F    Find
F3        Find Next 
Ctrl+K    (K)olor selector
Ctrl+D    Go to (d)efinition
Ctrl+I    Open file at cursor. (I)nclude.
Ctrl+E    Insert (e)cho: "echo"/"<?php echo ?>"
Alt+Up    Previous file
Alt+Down  Next file
Ctrl+Up   Previous function
Ctrl+Down Next function
Esc       Close auto-complete window
Alt+F4    Close program


Auto-complete:
-------------------------
objects,  "->"
classes,  "::"
arrays,   "['"
variables <by name>
functions <by name>
constants <by name>
keywords  <by name>


Features:
-------------------------
Function argument, javadoc preview
Tab in auto complete window inserts first common substring
